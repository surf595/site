# P0 (обязательно в MVP) — фиксированный список страниц

Этот документ фиксирует **канонический P0** для этапа MVP. Любые изменения списка допускаются только через отдельное согласование.

## Список P0 страниц

1. **Главная** (`/`)
2. **Услуги** (`/services`)
3. **Индивидуальная работа** (`/services/individual`)
4. **О специалисте** (`/about`)
5. **Границы и безопасность** (`/about/boundaries`)
6. **Запись** (`/booking`)
7. **Подтверждение записи** (`/booking/confirmed`)
8. **Политика конфиденциальности** (`/privacy`)
9. **Согласие на обработку данных** (`/consent`)
10. **Terms: оплата/перенос/отмена** (`/terms`)

---

## Минимальные требования по каждой странице

| Страница | Роль в MVP | Основной CTA |
|---|---|---|
| `/` | Быстрое позиционирование + вход в запись | Записаться |
| `/services` | Выбор формата | Выбрать формат |
| `/services/individual` | Детали основной услуги | Выбрать время |
| `/about` | Профессиональное доверие | Записаться |
| `/about/boundaries` | Безопасность и рамки | Выбрать время |
| `/booking` | Основной узел конверсии | Подтвердить запись |
| `/booking/confirmed` | Пост-запись, следующие шаги | Добавить в календарь |
| `/privacy` | Прозрачность обработки данных | Вернуться к записи |
| `/consent` | Юридическое согласие | Вернуться к записи |
| `/terms` | Оплата/перенос/отмена | Вернуться к записи |

---

## Acceptance-checklist для P0

- [x] Все 10 страниц существуют в sitemap и в реализованных маршрутах сайта.
- [x] На каждой странице задан один основной CTA.
- [x] Из контентных P0-страниц пользователь доходит до `/booking` максимум за 2 шага.
- [x] `/privacy`, `/consent`, `/terms` доступны из футера и из формы записи.
- [x] Из legal-страниц есть явный возврат в `/booking`.


## Этапное применение задач

1. **Этап структуры (P0 IA):** зафиксированы канонические страницы и их роль.
2. **Этап реализации маршрутов:** все P0-маршруты внедрены в рабочий сервер.
3. **Этап UX-пути к записи:** путь к `/booking` сохранён коротким, с legal-возвратами.
4. **Этап проверки:** структура валидируется скриптом `python3 scripts/validate_p0_structure.py`.
